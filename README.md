# start.bat-Server
Inicializador de Minecraft Server 
O start.bat é um arquivo txt que carrega um código que serve para ligar seu servidor de Minecraft;
Após eu inserir o código no txt, usando o bloco de notas, mudei o tipo do arquivo para .bat

Para você editar o contéudo do start.bat clique com o botão direito nele e abra com o bloco de notas, notepad++ ou algum aplicativo que você preferir.
No lugar do "meusotware-100.jar" cole o nome do arquivo da software de seu servidor, não esquecendo da .jar no final. Exemplo:
paper-1.21.3-83.jar
